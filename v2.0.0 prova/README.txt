v2.0.0

Cosa c'è di nuovo?

- E' stata implementata una struttura di tipo server-client, il client riceve solo le sue informazioni quindi è sicuro

- La comunicazione tra server e client è criptata usando l'algoritmo Fernet, la decrittazione è eseguita localmente

- Tutti i bug che rilevato durante lo sviluppo e i test sono stati risolti, ma non si sa mai, quindi se trovi qualche bug per favore dimmelo :)

What's new?

- It has been implemented a server-client platform, the client receive only his own information so is more secure

- The communication between the client and server is crypted using Fernet cryptography, the decryption is done locally

- Most of the bug had been resolved so it's stable but nobody knows so if you encounter a bug report it please :)
